# 🛡️ DiskHealth Monitor (Powered by CrystalDiskInfo)

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)
![Status](https://img.shields.io/badge/Status-Open%20Source-blue)

**DiskHealth Monitor** es una utilidad basada en el motor de CrystalDiskInfo para supervisar el estado de salud de tus unidades de almacenamiento (HDD, SSD, NVMe). Proporciona alertas tempranas sobre fallos de hardware mediante el análisis de datos S.M.A.R.T.

## ✨ Funciones Principales

* **Monitoreo S.M.A.R.T:** Visualiza errores de lectura, horas de encendido y sectores reasignados.
* **Alertas de Temperatura:** Controla el calor de tus unidades en tiempo real.
* **Gráficos de Evolución:** Sigue el desgaste de tus SSD (Health Status %).
* **Soporte Universal:** Compatible con discos externos USB y las últimas unidades NVMe.

## 🛠️ Requisitos del Sistema

| Requisito | Detalle |
| :--- | :--- |
| **Sistema Operativo** | Windows 7, 8, 10, 11 (32/64-bit) |
| **Almacenamiento** | < 10 MB de espacio libre |
| **Privilegios** | Requiere permisos de administrador para leer datos SMART |

## 🚀 Cómo empezar

1. Descarga el instalador desde el enlace superior.
2. Ejecuta el archivo y sigue los pasos de instalación.
3. Al abrirlo, verás un indicador de color:
    * **Azul/Verde:** El disco está sano.
    * **Amarillo:** ¡Precaución! Haz una copia de seguridad pronto.
    * **Rojo:** Riesgo inminente de fallo.
